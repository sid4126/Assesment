🛠 Setup Guide

Configure Supabase (database) and EmailJS (email delivery). Both free. ~10 minutes total.

SUPABASE — Step 1: Create project

Go to supabase.com → New Project → Settings → API → copy Project URL and anon public key.

SUPABASE — Step 2: Create table (SQL Editor)
create table assessments (
  id          bigint generated always as identity primary key,
  created_at  timestamptz default now(),
  name        text, email text, age int, department text,
  temperament text, mbti_code text,
  score_ei int, score_sn int, score_tf int, score_jp int,
  pct_ei int, pct_sn int, pct_tf int, pct_jp int
);
SUPABASE — Step 3: Add credentials

Replace YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON_KEY in the script below.

EMAILJS — Step 1: Create account

Go to emailjs.com → Sign up → Account → API Keys → copy Public Key.

EMAILJS — Step 2: Connect email service

Email Services → Add New Service → connect Gmail or Outlook → copy Service ID.

EMAILJS — Step 3: Create template

Set the "To Email" field to {{to_email}} — this is critical. Use these variables in the body:

Hi {{user_name}},

Your Temperament: {{temperament_name}} ({{mbti_code}})
{{tagline}}

SCORES
E/I: {{score_ei}} | S/N: {{score_sn}}
T/F: {{score_tf}} | J/P: {{score_jp}}

{{description}}

STRENGTHS
{{strengths}}

GROWTH AREAS
{{weaknesses}}

CAREER FIT: {{careers}}
Department: {{department}} | Age: {{age}}
EMAILJS — Step 4: Add credentials

Replace YOUR_EMAILJS_PUBLIC_KEY, YOUR_EMAILJS_SERVICE_ID, YOUR_EMAILJS_TEMPLATE_ID in the script.
